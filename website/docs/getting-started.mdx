# Getting started

Install heph:

```bash title="terminal"
curl -fsSL https://hephbuild.github.io/install.sh | sh
```

Then drop a `.hephconfig` at the root of your repository. Pin the version so
every machine and CI run resolves the same toolchain — byte for byte:

<GettingStartedHephConfig />

The version above is resolved live from the latest heph release. From here,
declare targets in `BUILD.heph` files and run:

```bash title="terminal"
heph build //...
```
